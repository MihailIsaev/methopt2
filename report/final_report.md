## Формулы моделей (пакет 3)

Обозначения: `A` — матрица признаков размера `m×n`, `x` — вектор параметров, `z = Ax`.

### Pseudo-Huber (регрессия)

`L_i = sqrt(1 + ((z_i - y_i)/δ)^2)`  
`f(x) = (1/m) Σ_i L_i + (λ/2)||x||^2`

Пусть `r = Ax - y`, `s = sqrt(1 + (r/δ)^2)`. Тогда

`∇f(x) = (1/m) A^T ( r / (δ^2 s) ) + λx`

`H(x)v = (1/m) A^T ( (Av) / (δ^2 s^3) ) + λv`

### Smoothed SVM (классификация)

`L_i = log(1 + exp(1 - y_i z_i))`, `y_i ∈ {±1}`  
`f(x) = (1/m) Σ_i L_i + (λ/2)||x||^2`

Пусть `t = 1 - y ⊙ z`, `p = sigmoid(t)`. Тогда

`∇f(x) = (1/m) A^T ( - y ⊙ p ) + λx`

`H(x)v = (1/m) A^T ( (p ⊙ (1-p)) ⊙ (Av) ) + λv`
